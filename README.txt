Scoring Transponder Arduino Sketch

For MW Transponder 2013 R-TEAM Version or MW Transponder 2017
Load the Arduino Duemilanove or Uno bootloader with ISP

For MW Transponder 2013 load hex file with ISP

Not compatible with 2012 MW Transponders or older.

To build, download the following libraries and install in your Arduino add-on libraries:

https://github.com/PaulStoffregen/TimerOne
https://github.com/GreyGnome/PinChangeInt
